Programs for calculating the thermocouple reference function and its inverse

Python Implementation

Package tc_python.zip (download)
This package consists of:
  • the implementation of the reference function and the inverse function
  • demo and sample tests
  • sample data obtained from the NIST database
  • sample test results
External libraries used by this package may be downloaded separately if not included in this distribution (see Libraries).
Reference Function Fn.py
This class consists of methods to:
  • load the parameters of a reference function in JSON format
  • compute the EMF for a given temperature T
  • compute the derivative of EMF with respect to a given temperature T
Inverse Function FnInv.py
This class consists of methods to:
  • load the parameters of an inverse function in JSON format
  • compute the temperature T for a given EMF
Demo demo.py
This script consists of a demo on how to apply the functions to a basic circuit.
Tests test.py
This script consists of a collection of sample tests on the values of F and Finv. The results of these sample tests may be viewed at: Python test results
Platforms This package has been compiled and tested under Python version 2.7
